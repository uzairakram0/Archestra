# Archestra

Architectural design studio website — producing 2D construction plans and 3D visualizations for new builds, additions, and renovations across Texas and New York.

## Structure

```
Archestra/
├── index.html        # HTML structure
├── css/
│   └── styles.css    # All styles
├── js/
│   ├── main.js       # Wizard, calendar, modals, theme toggle
│   └── hero-3d.js    # Three.js wireframe building animation
└── assets/           # Images and media
```

## Features

- Multi-step inquiry wizard with scheduler and project estimator
- Interactive 3D wireframe building (Three.js) in the hero
- Light / dark mode with system preference detection
- Project portfolio with modal popups
- Fully responsive — mobile through desktop
- No framework, no build step — plain HTML, CSS, and JavaScript

## Deployment

Drop the project folder as a zip into [Netlify](https://netlify.com), or connect the GitHub repo for automatic deploys on every push.
